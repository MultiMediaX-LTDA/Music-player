# 🎧 Jammer

Seu player, seu deck. Jam e play.

> "Your player, your own deck. Jam and play." -- Jammer, the moon jellyfish

## Sobre
Jammer é um music player FOSS com filosofia Linux-alike: alta customização, permissões mínimas, zero rastreamento, e código aberto sob licença FOSS MultMediaX.

## Estado Atual
- ✅ Estrutura Gradle + GitHub Actions
- ✅ CrashReporter (privacy-respecting)
- ✅ Dependências configuradas (ExoPlayer, Shizuku, Retrofit)
- 🔄 Em desenvolvimento: player funcional

## Compilar
O GitHub Actions builda automaticamente a cada `push` na branch `main`.  
Baixe o APK em: **Actions → Build Jammer APK → Artifacts**
